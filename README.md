This repository contains a bioinformatics analysis of gene expression signatures and pathway dysregulation in Sjögren’s disease using the GSE51092 microarray dataset. The study includes differential gene expression analysis, unsupervised exploration (PCA, UMAP, K-means clustering), supervised machine learning models (Random Forest, LDA, LASSO logistic regression), and functional enrichment analysis using g:Profiler and PathfindR. The results highlight a strong interferon-driven transcriptional signature and identify potential biomarkers associated with Sjögren’s disease.
